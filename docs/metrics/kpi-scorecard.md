# KPI Scorecard

## Activation
- First-trip-completed rate within first session window.
- Document upload completion rate.
- First itinerary generation rate.

## Retention
- D7 and D30 retained active travelers.
- Weekly active coordinators.
- Notification usefulness ratio (actioned/muted).

## Revenue
- Free to paid conversion.
- Team expansion (seat growth + AI top-ups).
- Affiliate conversion quality by provider.

## Reliability
- API p95 read/write latency.
- Error rate by endpoint.
- Availability for core dashboard flows.
