# Core User Flows

## Flow 1: Trip Creation
1. Upload confirmation email/PDF.
2. Parse and structure trip card.
3. Suggest hotel zones and itinerary draft.
4. Assign coordinator and traveler notifications.

## Flow 2: Disruption Handling
1. Decision Strip shows status change.
2. User opens "Why changed" trust context.
3. Coordinator re-optimizes itinerary and updates handoff notes.
4. Alert sent to traveler and stakeholders.

## Flow 3: Team Oversight
1. Coordinator opens Team Board.
2. Sorts by disruption risk and departure window.
3. Resolves high-risk trips with delegated actions.
