# Design Principles

1. Anxiety-first hierarchy: show the next action before secondary data.
2. Progressive disclosure: default condensed cards, expand for depth.
3. Trust by transparency: show source, freshness, and confidence.
4. Team-first utility: prioritize disruption risk and delegation controls.
5. Motion with purpose: animation highlights change, not decoration.
6. Accessibility baseline: semantic HTML, contrast-aware palettes, keyboard support.
