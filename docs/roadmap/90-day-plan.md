# 90-Day Plan

## Days 0-30
- Complete design system and core route architecture.
- Implement auth context, role model, trip schema, and API contracts.
- Ship dashboard prototype for user testing with business travelers/coordinators.

## Days 31-60
- Launch working command center: live status, itinerary, hotel discovery, document vault.
- Add billing, AI credits, and affiliate tracking.
- Instrument KPI scorecard and operational dashboards.

## Days 61-90
- Harden enterprise controls: RBAC policies, audit UI, and security trust center.
- Run retention and onboarding experiments.
- Finalize iOS API readiness and mobile integration backlog.
